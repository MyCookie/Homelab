# Homelab

Config for a self-hosted homelab: Docker Compose stacks for Nextcloud, Jellyfin, Synapse, Forgejo, and other services, a Caddy reverse-proxy setup, and operational docs for running it all on Debian + ZFS.

Evolved from [docker_cloud](https://github.com/MyCookie/docker_cloud) (2019–2021), which ran the same idea on Docker Swarm + nginx before this repo moved to Compose, Caddy, ZFS, and a GPU LLM-inference layer.
